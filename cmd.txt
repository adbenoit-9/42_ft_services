Start a cluster:
OS :
brew install hyperkit
minikube start --vm-driver=hyperkit
or
minikube start --vm-driver=docker
VM :
minikube start --vm-driver=docker

Check if kubectl is installed;
kubectl version

View cluster details:
kubectl cluster-info

View the nodes (aka. esclaves) in the cluster:
kubectl get nodes

kubectl get pod
kubectl get services

Create deployment:
kubectl create deployment NAME --image=image
ex: ubectl create deployment nginx-depl --image=nginx
kubectl get pod

kubectl get replicaset

Edit deployment:
kubectl edit deployment NAME
=> config file

kubectl logs POD_NAME

Create a config file :
touch config-file.yaml => add config
kubectl apply -f config-file.yaml

(1) kubectl decribe service name-service
(2) kubectl get pod -o wide

Verify if is on the good port:
compare entrypoint of (1) and the IP of (2)

Get more info on the deployement: 
kubectl get deployement name-deployement -o yaml > name-deployement-result.yaml
